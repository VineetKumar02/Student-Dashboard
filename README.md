# Student-Dashboard
This is a MEAN stack project for managing student details.

#Installation
Clone the repository
Navigate to the project directory
Run npm install to install the dependencies
Run npm start to start the server


#Dependencies
body-parser: 1.20.2
cors: 2.8.5
express: 4.18.2
mongodb: 5.3.0
mongoose: 7.0.4
nodemon: 2.0.22
